使用说明

  1. 先点击右侧 "获取 Snapshot" 按钮获取元素 ID
  2. 在 "添加注释" 输入框中填写元素 ID（如 e1）和注释内容
  3. 选择位置（下拉菜单）和样式配色
  4. 点击 "添加注释" 查看效果
  5. 点击 "清除注释" 可移除所有标注
