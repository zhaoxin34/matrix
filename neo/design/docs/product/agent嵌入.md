## 🎯 产品概述

### 目标系统定义

- 目标系统是 PC 端的 HTML 系统
- 目标系统允许嵌入 Agent

### 业务目标

通过把我们设计的 Agent 嵌入到被学习、操作的系统中，实现如下目标：

- 采集用户动作、系统数据
- 操作目标系统

---

## 👤 用户角色与故事

### 用户角色

| 角色     | 描述                         |
| -------- | ---------------------------- |
| 新手员工 | 不熟悉系统操作，需要引导     |
| 资深员工 | 熟练使用系统，可以训练 Agent |
| 一般员工 | 给 Agent 设定目标，执行动作  |

### 故事一：引导模式

张三是个新入职的员工，不了解软件该如何工作。点击 Agent 图标，选择**引导模式**，Agent 调出曾经高手的操作记录，边演示边解释操作内容。期间张三可以随时打断操作，抛出问题，Agent 给予解答。

### 故事二：学习模式

李四是一位非常资深的员工。点击 Agent 图标，选择**学习模式**，Agent 开始学习、评估、预测李四的行为。到了困惑的时候可以向李四提问，为什么要那样操作。

### 故事三：主动模式

给 Agent 设定明确的目标，Agent 开启**主动模式**后，按照目标主动操作软件。用户可以实时监控 Agent 的操作过程，必要时接管。

## 🎨 Agent UI 设计

### 3.1 形态与布局

#### 浮动 Icon

| 属性 | 说明                              |
| ---- | --------------------------------- |
| 位置 | 屏幕角落（右下角默认，可拖拽）    |
| 样式 | 轻量图标，类似 Siri 交互体验      |
| 状态 | 空闲 / 工作中 / 异常（见第 5 章） |

#### 模式切换菜单

点击 Icon 后弹出菜单，包含三个选项：

- 🏫 学习模式
- 🎬 引导模式
- 🚀 主动模式

### 3.2 视觉规范

#### 轻量浮层风格

采用轻量浮层风格，确保不遮挡主体业务内容：

- 半透明背景（rgba 白色，opacity 0.95）
- 圆角（12px）
- 阴影（box-shadow: 0 4px 20px rgba(0,0,0,0.1)）
- 毛玻璃效果（backdrop-filter: blur(10px)）

#### 遮罩层视觉规范

引导模式下使用全屏遮罩：

- 背景色：rgba(0, 0, 0, 0.6)
- 操作区域：高亮显示（边框 + 光晕效果）
- Agent 文字输出区域：固定在遮罩上方，白色卡片样式

### 3.3 交互规范

#### 核心交互流程

```
用户点击 Icon
    ↓
弹出模式选择菜单
    ↓
用户选择模式（学习/引导/主动）
    ↓
进入对应模式状态
    ↓
用户可随时中断（点击遮罩外部 / 快捷键）
    ↓
恢复或退出
```

#### 中断与恢复

- **引导模式中断**：点击遮罩外部或按 ESC，暂停录像播放
- **学习模式中断**：对话过程中随时输入问题
- **主动模式中断**：用户可随时接管操作

## ⚙️ 功能设计

### 4.1 学习模式

| 功能       | 描述                                             |
| ---------- | ------------------------------------------------ |
| 触发与退出 | 点击 Icon → 选择学习模式 → 开始记录              |
| 操作记录   | 记录用户在系统中的所有操作（点击、输入、滚动等） |
| 行为评估   | Agent 分析用户行为模式，识别操作意图             |
| 行为预测   | 基于历史数据预测用户下一步操作                   |
| Chat 交互  | 用户可随时打断，提问，Agent 给予解答             |

#### 交互说明

- Agent 开始学习后，Icon 显示"学习中"状态
- 用户操作时，Agent 实时记录并分析
- 当 Agent 困惑时，弹出问题卡片，等待用户回答
- 学习完成后，Agent 具备预测该用户行为的能力

#### 边界情况

- 用户长时间无操作：暂停记录，超时提醒
- 网络中断：本地缓存，恢复后同步
- 权限不足：提示用户，Agent 跳过该操作

---

### 4.2 引导模式

| 功能       | 描述                               |
| ---------- | ---------------------------------- |
| 录像选择   | 根据当前页面上下文，推荐相关录像   |
| 录像播放   | 播放 rrweb 录制的操作录像          |
| 遮罩同步   | 遮罩层与录像播放同步，遮挡真实系统 |
| Agent 讲解 | Agent 输出文字，解释当前操作       |
| 用户打断   | 用户随时提问，Agent 暂停并解答     |
| 操作演示   | 在遮罩上高亮当前操作区域           |

#### 交互说明

```
进入引导模式
    ↓
加载相关录像
    ↓
显示全屏遮罩（遮挡真实系统）
    ↓
播放录像 + Agent 文字输出
    ↓
用户可随时点击遮罩提问
    ↓
Agent 暂停并回答
    ↓
继续播放或用户主动结束
```

#### 遮罩与录像同步

- 录像每帧对应遮罩状态
- 操作区域使用高亮边框 + 微光效果
- Agent 文字输出固定在顶部或底部

#### 边界情况

- 录像加载失败：提示用户，提供重试选项
- 录像与当前页面不匹配：提示版本差异，询问是否继续
- 用户快速点击跳过：尊重用户选择，但记录跳过行为

---

### 4.3 主动模式

| 功能     | 描述                                 |
| -------- | ------------------------------------ |
| 目标设定 | 用户输入目标（如"帮我创建一个客户"） |
| 操作分解 | Agent 将目标分解为具体步骤           |
| 操作执行 | Agent 直接在页面上执行操作           |
| 实时可见 | 所有操作对用户可见，支持监控         |
| 用户接管 | 用户可随时接管，停止 Agent           |
| 异常处理 | 操作失败时，提示用户并请求指令       |

#### 交互说明

```
用户设定目标
    ↓
Agent 分析当前页面状态
    ↓
Agent 展示操作计划（用户确认）
    ↓
开始执行（每步操作高亮可见）
    ↓
用户可随时接管或终止
    ↓
完成后展示总结
```

#### 操作高亮

- 当前操作元素使用脉冲边框高亮
- 即将操作的元素使用虚线边框预览
- 操作完成后短暂显示成功标识

#### 边界情况

- 页面状态变化：Agent 暂停，重新评估
- 权限不足：提示用户，等待授权
- 操作超时：提示失败原因，提供替代方案

---

### 4.4 Chat 能力（通用）

| 功能       | 描述                             |
| ---------- | -------------------------------- |
| 对话界面   | 浮层面板内，包含输入框和消息列表 |
| 打断机制   | 任何模式下随时呼出 Chat          |
| 上下文关联 | Chat 理解当前操作上下文          |
| 恢复机制   | Chat 结束后恢复到之前状态        |

#### 界面布局（待 UI 设计阶段细化）

- 消息列表：显示历史对话
- 输入框：支持文本输入
- 快捷卡片：常见问题快速触发

## 📊 Agent 状态设计

### 状态类型

| 状态   | 图标变化     | 说明           |
| ------ | ------------ | -------------- |
| 空闲   | 灰色静态图标 | 等待用户交互   |
| 学习中 | 蓝色脉冲图标 | 记录用户操作中 |
| 引导中 | 橙色播放图标 | 播放录像演示中 |
| 执行中 | 绿色旋转图标 | Agent 正在操作 |
| 异常   | 红色警告图标 | 需要用户关注   |

### 状态可视化

- Icon 颜色变化
- 悬浮提示（鼠标悬停显示状态描述）
- 点击 Icon 弹出状态详情面板

## 🚫 非工作范围

### 明确不包括

- Agent 设计本身（如何学习、如何分配给员工）
- 录像的录制与管理（假设 rrweb 录像已存在）
- 目标系统的后端改造

### 未来考虑

以下内容暂不在范围内，但以后可能会考虑：

- 移动端适配
- 多语言支持
- Agent 自我优化机制
- Agent 安全的机制
