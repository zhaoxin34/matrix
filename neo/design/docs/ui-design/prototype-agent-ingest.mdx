# Agent 嵌入原型

点击 Agent Icon 后弹出模式菜单，可选择进入学习、引导或主动模式。

**[查看 Agent 嵌入原型 →](/prototype/agent-ingest)**

## 交互说明

| 步骤 | 操作                  | 结果                                |
| ---- | --------------------- | ----------------------------------- |
| 1    | 点击右下角 Agent Icon | 弹出模式选择菜单                    |
| 2    | 选择学习模式          | Icon 变为蓝色脉冲状态，显示学习面板 |
| 3    | 选择引导模式          | 全屏遮罩 + rrweb 录像播放           |
| 4    | 选择主动模式          | 弹出目标设定面板                    |

## 设计要点

- **浮动 Icon**：固定在右下角，可拖拽，悬浮显示状态
- **模式菜单**：点击后弹出，带动画效果
- **状态反馈**：Icon 颜色和动画表示当前状态
