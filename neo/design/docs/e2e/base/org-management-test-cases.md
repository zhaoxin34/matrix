# Test Cases: 组织管理

## 概述

- **功能模块**: 组织管理（Organization Management）
- **需求来源**: `design/docs/product/admin/组织管理设计.md`
- **测试地址**: localhost:3300/admin/org-structure
- **测试覆盖**: 组织 CRUD、员工管理、状态转换、删除约束
- **最后更新**: 2026-05-27

---

## 1. 功能测试

### TC-F-ORG-001: 查看组织树形结构

- **需求**: 组织管理设计 - 2.1 查看组织
- **优先级**: High
- **前置条件**:
  - 超级管理员已登录系统
  - 系统存在组织数据（至少包含公司级别组织）
- **测试步骤**:
  1. 访问 `/admin/org-structure` 页面
  2. 观察左侧组织结构树
- **预期结果**:
  - 组织树以层级结构展示
  - 根组织显示在公司/企业级别
  - 子组织缩进显示在父组织下方
  - 每个组织节点显示成员数量
  - 展开/折叠按钮正确显示（有无子组织）

- 重新入职需新建档案
- 软删除（`is_deleted=true`）的员工**不计入**任何校验

- 调动为**直接调动**，无需审批流程
- **任何人都可以发起调动**
