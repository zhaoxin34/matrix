设计系统 · 实现要点

这套系统是怎么长在一起的。

一个强调色，只用两处。

电光蓝只留给用户消息和加载态。Agent 气泡保持中性灰，错误用专属红色。绝不靠紫色渐变凑数。

贴角的几何。

FAB 距右下角 24px，对话框沿同一锚点叠在上方，距底部 96px。收起时直接回到那一点——不做滑出销毁的动画。

错误用内联卡片，绝不弹模态。

选择器没命中或超时，会以一张带 Retry chip 的浅色卡片呈现。没有全屏遮罩，没有静默失败。即使 agent 卡壳，语感仍是对话式的。
