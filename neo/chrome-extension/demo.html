🎯 SteerButton 组件

这是一个浮动圆形按钮组件，带有炫酷的边框发光动画效果。 点击按钮可以打开模式选择菜单，切换 Agent 的运行模式。

🎨

边框发光

彩虹色渐变边框，旋转动画

🔄

呼吸效果

脉冲式发光，明暗交替

⚡

模式切换

学习/指导/主动三种模式

使用说明：

请在 Chrome 中加载扩展后，访问任意网页。浮动按钮将出现在右下角。

组件代码位置: src/components/SteerButton.tsx

📋 功能列表

  • ✅ 浮动按钮 - 固定在页面右下角，不影响页面内容
  • ✅ 发光边框 - 使用 CSS @property 实现彩虹渐变旋转
  • ✅ 脉冲动画 - 录制状态时的红色脉冲效果
  • ✅ 模式指示 - 右上角小圆点显示当前模式
  • ✅ 下拉菜单 - 点击展开模式选择，ESC 或外部点击关闭

🎨 设计参考

组件样式参考了 border-beam 库的视觉效果：

  • 使用 conic-gradient 创建锥形渐变边框
  • 使用 @property 实现可动画的自定义属性
  • 使用 mask 和 mask-composite 实现镂空边框效果
  • 多重阴影叠加实现发光效果
