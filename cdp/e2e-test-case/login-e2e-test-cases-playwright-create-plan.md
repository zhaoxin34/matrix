### 发现的通用问题

1. **Element Refs 动态变化**: 页面每次加载后element refs都会发生变化，脚本中硬编码的refs(如e37, e48等)在后续交互后会失效
   - 解决方案: 使用playwright-cli的fill/click方法时传入选定的element ref

2. **浏览器进程问题**: playwright-cli在使用持久化context时多次出现浏览器崩溃
   - 解决方案: 每次测试使用独立的浏览器session（isolated: true）

3. **后端验证问题**:
   - 错误密码登录成功 (CDP-LOG-002)
   - 不存在用户登录成功 (CDP-LOG-007)
   - 这些问题需要后端修复

4. **Socket地址冲突**: playwright-cli的socket文件有时未正确清理，导致EADDRINUSE错误
   - 解决方案: 使用 `isolated: true` 配置避免冲突
