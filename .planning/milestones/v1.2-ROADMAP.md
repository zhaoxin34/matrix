# Milestone v1.2: Shopping Cart & Checkout

**Status:** ✅ SHIPPED 2026-04-20
**Phases:** 6-7
**Total Plans:** 4

## Overview

Shopping Cart and Order Management for the AI Matrix e-commerce platform. This milestone added:
- Guest and authenticated user cart support via session cookies
- Add-to-cart with SKU variant selection and stock warnings
- Checkout flow with address management
- Order history and order detail pages

## Milestone Summary

**Key Decisions:**
- Guest users identified by session_id cookie, authenticated users by JWT user_id
- Cart items can have either user_id OR session_id (nullable ForeignKey)
- Merge on login: if user already has product, add quantities; otherwise transfer ownership
- Backend handles cart-to-order conversion internally, frontend sends address only
