## Pages & Components

### 1. Product List Page (`/products`)

**Layout:** Left sidebar (240px) + main content

| Element | Specification |
|---------|---------------|
| Sidebar | Category tree (L1 accordion, L2 nested) |
| Search Bar | Top of main content, size="large", placeholder "搜索商品..." |
| Autocomplete | Dropdown below search, max 10 items, 300ms debounce |
| Filter Section | Below search: Brand (multi-select), Price Range (min/max), Stock (switch) |
| Sort Bar | Below filters: Relevance, Price Low→High, Price High→Low, Sales, Newest |
| Product Grid | 4 columns (md), 3 columns (sm), 2 columns (xs) |
| Product Card | Image (200px height), name, price, stock badge |
| Pagination | Bottom of grid, default 20 items per page |

**Interactions:**
- Category click filters products (updates URL query param)
- Search triggers after 300ms debounce
- Autocomplete shows top 10 matching product names
- Filter changes trigger immediate API call
- Sort changes trigger immediate API call

### 2. Product Detail Page (`/products/:id`)

**Layout:** Two-column (image left 60%, details right 40%)

| Element | Specification |
|---------|---------------|
| Image Carousel | Main image (400x400) + thumbnail strip, up to 8 images |
| Product Name | `<Title level={2}>{name}</Title>` |
| Price Display | Original price (strikethrough if on sale) + sale price in red |
| Stock Status | "有货" (green), "仅剩X件" (orange), "缺货" (red) |
| SKU Selector | For variants like color/size, dropdown or button group |
| Quantity | `<InputNumber min={1} max={stock} />` |
| Add to Cart | `type="primary"`, size="large", "加入购物车" |
| Specifications | Table of product specs |
| Description | Expandable text area |

**Interactions:**
- SKU selection updates price and stock
- Quantity input validates against stock
- Add to Cart shows success message + updates cart badge

## Component States

### Product Card

| State | Visual |
|-------|--------|
| Default | White background, subtle shadow |
| Hover | Elevated shadow, slight scale (1.02) |
| Out of Stock | Grayscale image, "缺货" badge overlay |

| State | Visual |
|-------|--------|
| Loading | `<Spin size="small" />` in dropdown |
| Results | List of product names, highlighted match |
| No Results | "未找到相关商品" |
| Error | "搜索失败，请重试" |

| Element | Specification |
|---------|---------------|
| Brand | `<Select mode="multiple">` with brand options |
